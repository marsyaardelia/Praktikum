About Smart waste scanner


This application is a smart technology–based platform designed to help communities manage waste in a more practical and efficient way. Its main feature is a smart scanner that can identify and differentiate between organic and non-organic waste using a smartphone camera, making it easier for users to sort their trash correctly.


In addition, the app provides a waste collection reporting service, allowing users to report delayed pickups or waste accumulation so that authorities can respond quickly. To keep users engaged, the application also includes educational games that teach environmental awareness in a fun and interactive way.



Overall, the app aims to increase public awareness and participation in creating a cleaner and more sustainable environment.
